# GemCollector
Collect gems used by applications.

## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'gem_collector'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install gem_collector
```
